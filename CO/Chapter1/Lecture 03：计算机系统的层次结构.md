# 硬件实现和软件实现

|       |     软件实现      |               硬件实现                |
| :---: | :-----------: | :-------------------------------: |
| 性能与速度 |     比硬件低      |               更高更快                |
|  灵活性  |  相对容易进行修改和更新  |             需要重新设计和制造             |
|  成本   |   开发软件相对便宜    |            开发和制造硬件比较昂贵            |
| 应用需求  | 逻辑不太复杂，性能要求不高 | 由于电路可以并行执行操作，<br>一些高频的交易系统在硬件层面完成 |
